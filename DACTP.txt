üéØ What DACTP Does (Protocol Definition)

DACTP is a decentralized trust and delegation layer for Stellar/Soroban.

It provides:
1) On-chain reputation as a first-class primitive
2) Revocable, scoped agent delegation
3) Deterministic reputation updates based on financial behavior
4) A reusable protocol that any Stellar dApp can integrate

DACTP does NOT:
- Issue tokens
- Perform lending itself
- Perform KYC
- Make subjective trust decisions

It only enforces rules.

üí° Reputation Update Logic (MVP-Level)

Reputation updates are triggered ONLY by financial outcomes.

Positive event:
Loan repayment completed
‚Üí +5 reputation

Negative event:
Loan default / missed repayment
‚Üí ‚àí15 reputation

Severe event:
Fraud / malicious behavior
‚Üí reputation frozen (no more delegation)

Key rule:
Reputation increases slowly and decreases quickly.
